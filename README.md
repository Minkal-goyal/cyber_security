# AI Network Traffic Classifier

An AI-based Network Traffic Classification project using Machine Learning techniques to detect and classify network traffic patterns from the KDD Cup 99 dataset.

---

## 📌 Project Overview

This project uses machine learning algorithms to analyze and classify network traffic data. It performs complete data preprocessing, feature selection, model training, and evaluation to identify network traffic behavior efficiently.

The project is built using Python and popular machine learning libraries such as Scikit-learn, Pandas, NumPy, and Matplotlib.
